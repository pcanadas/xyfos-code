# Xyfos Code

Landing + blog + changelog + roadmap for Xyfos Code.

## Local development

```bash
npm ci
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy

This repo deploys `dist/` to `xyfos.com` via GitHub Actions over SSH.
